# ITパスポートOne 利用規約

この利用規約（以下、「本規約」とします）は、スマートフォン向けアプリケーション **「ITパスポートOne」**（以下、「本アプリ」とします）の利用条件を定めるものです。  
ユーザーは、本アプリを利用することにより、本規約に同意したものとみなされます。

---

## 第1条（適用）

本規約は、ユーザーと開発者（前野太洋 / SunProject）（以下、「当方」とします）との間の、本アプリの利用に関わる一切の関係に適用されるものとします。

---

## 第2条（AI機能の利用と免責）

1. 本アプリの「AI問題生成機能」および「AI解説機能」は、Google LLC が提供する生成AIサービス（Gemini API 等）を利用しています。  

2. AIによって生成される問題や解説は、必ずしも正確性、完全性、最新性を保証するものではありません。  
   ユーザーは、生成された内容を自身の判断と責任において利用するものとします。

3. ユーザーは、本機能を利用して以下の内容を含むデータをアップロードしてはならないものとします。

   - 著作権等の第三者の権利を侵害する画像・資料  
   - 公序良俗に反する内容  
   - 法令に違反する内容  

4. 当方は、ユーザーがアップロードしたデータおよびAIによって生成された内容について、その正確性や適法性を保証するものではありません。

---

## 第3条（有料機能・サブスクリプション）

1. 本アプリの一部機能（広告非表示、AI機能、クラウド同期など）は、有料のサブスクリプション（定期購読）として提供されます。

2. サブスクリプションの利用料金は、ユーザーの **Apple ID アカウント** に請求されます。

3. 購読期間終了の **24時間前までに解約設定を行わない限り、サブスクリプションは自動的に更新** されます。

4. 解約手続きは、端末の設定アプリからユーザー自身で行う必要があります。
